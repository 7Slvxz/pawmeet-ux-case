# 🐾 PawMeet — Plataforma de Adoção de Animais

O **PawMeet** é uma solução digital desenhada para conectar protetores independentes, ONGs e abrigos de animais a potenciais adotantes de forma simples, segura e humanizada. O projeto foca em solucionar os gargalos de comunicação e burocracia que hoje dificultam o processo de adoção no Distrito Fedral

---

## 🎯 O Desafio & Objetivo
Muitos abrigos enfrentam dificuldades para gerenciar o fluxo de adoções, enquanto adotantes enfrentam processos burocráticos ou falta de informações claras sobre os animais. O objetivo do PawMeet é criar uma ponte amigável, centralizando catálogos de animais, agendamento de visitas e chats de suporte.

---

## 👥 Pesquisa & Personas (UX Research)
Para entender o cenário real, foram desenvolvidas pesquisas com base em perfis de usuários do ecossistema:
*   **Cláudia (Gestora de Abrigo):** Precisa de uma ferramenta eficiente para cadastrar animais e triar adotantes sem perder tempo em processos manuais.
*   **Samuel (Adotante):** Busca uma plataforma confiável e intuitiva para encontrar seu novo companheiro e tirar dúvidas sobre o processo em Brasília.

## 📊 Arquitetura e Engenharia de Software
Para garantir a viabilidade técnica do sistema, o projeto conta com documentação e modelagem arquitetural:
*   **Mapeamento de Processos (BPMN):** Fluxos de triagem de adoção e agendamento de visitas.
*   **Diagrama de Casos de Uso:** Definição das permissões de Visitantes, Adotantes e Administradores (ONGS).
*   **Diagrama de Classes (UML):** Estrutura de dados pensada para escalabilidade do banco de dados.

---

## 🚀 Próximos Passos (Roadmap de TI)
Atualmente, o projeto está na fase de **Design e Especificação Técnica**. O planejamento para o desenvolvimento envolve:
1.  Criar o Banco de Dados Relacional (Modelagem SQL).
2.  Desenvolver a API Backend (utilizando Python/FastAPI ou Java).
3.  Codificar o Frontend responsivo integrando com o protótipo do Figma.
